Cosmic Survivor

A 2D platformer game inspired by Broforce, Terraria, Mario, and Hollow Knight

## Installation

1) Clone this repository:

```bash
git clone https://github.com/jlk31/CS-NEA/git

2) Install dependencies:

cd CS-NEA
pip install -r requirements.txt

3) Run the game

python src/main.py

```
